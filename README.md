O try-catch serve pra evitar que o programa trave quando acontece um erro.
No try eu coloco o código que pode dar problema, e no catch eu trato esse erro, tipo mostrando uma mensagem ou fazendo outra coisa no lugar.

ele previne o estrago que o erro causaria. Sem um bom try-catch o programa quebraria facilmente.
em resumo Try-catch não necessariamente evita o erro, mas evita que o programa pare por causa dele.
